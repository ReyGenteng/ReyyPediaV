Authentication

All API requests require authentication using an API key. The API key can be provided in two ways:

Header Authentication

Include your API key in the request header as X-APIKEY

Query Parameter

Add your API key as a query parameter apikey

Security Features

Our API includes advanced security features to protect your account and ensure secure access.

Security Measures

Feature Description Status
IP Whitelisting Restrict API access to specific IP addresses Active
Rate Limiting 5-second cooldown between requests Active
Account Verification Verified accounts only Required

User Profile

Retrieve comprehensive user profile information including account details, balance, and verification status.

GET https://ciaatopup.my.id/h2h/profile

Get User Profile

Retrieves the authenticated user's profile information including personal details, account balance, and verification status.

Request Example
curl -X GET "https://ciaatopup.my.id/h2h/profile" \ -H "X-APIKEY: your_api_key_here"
Request Example
const response = await fetch('https://ciaatopup.my.id/h2h/profile', { method: 'GET', headers: { 'X-APIKEY': 'your_api_key_here', 'Content-Type': 'application/json' } }); const data = await response.json(); console.log(data);
Request Example
import requests url = "https://ciaatopup.my.id/h2h/profile" headers = { "X-APIKEY": "your_api_key_here", "Content-Type": "application/json" } response = requests.get(url, headers=headers) data = response.json() print(data)

Panel Services

Create Panel Service

Creates a new hosting panel service with specified resource configuration and automatically sets up user account and server.

Query Parameters

Parameter Type Required Description
username string Required Username for the panel account
paket string Required Package type (1gb, 2gb, 3gb, 4gb, 5gb, 6gb, 7gb, 8gb, 9gb, 20gb, unli)

Available Packages

Package Memory Disk CPU Price (User) Price (Reseller)
1gb 1024 MB 1024 MB 50% Rp 1,100 Rp 1,050
2gb 2048 MB 2048 MB 100% Rp 2,200 Rp 2,100
unli Unlimited Unlimited Unlimited Rp 16,500 Rp 15,750
Request Example
curl -X GET "https://ciaatopup.my.id/h2h/order-panel?username=johndoe&paket=2gb" \ -H "X-APIKEY: your_api_key_here"
